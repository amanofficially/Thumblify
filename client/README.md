# 🚀 Thumblify

> AI-powered YouTube Thumbnail Generator built with React + Vite ⚡  
> Create high-CTR thumbnails in seconds with a modern UI.

---

## ⚡ Tech Stack

- React ⚛️
- Vite ⚡
- Tailwind CSS 🎨
- React Router 🔀
- Framer Motion ✨

## ✨ Features

- 🎯 AI-powered thumbnail generator UI
- ⚡ Super fast Vite setup
- 📱 Fully responsive design
- 🎬 YouTube preview system
- ✨ Smooth animations
- 🧠 CTR-focused design system
